def run_task( self, file_name: str, files_path: str | None = None, worker_config: practicuscore.WorkerConfig | str | dict | None = None, terminate_on_completion: bool = True, capture_task_output: bool = True, python_venv_name: str | None = None, max_files_to_upload: int = 250, *args, **kwargs) -> tuple[practicuscore.worker_manager.Worker | None, bool]:

Runs a task on a 'remote worker'. A task can be any type of code, including Practicus AI generated. Please do not call this method to run code on current worker.

Parameters
  • file_name: A .py python or .sh script file to run on the worker
  • files_path: Directory of all the files to upload to remote worker, code files, configuration files. Leave empty for current directory.
  • worker_config: Remote worker configuration
  • terminate_on_completion: Terminate the worker that was created to run the task.
  • capture_task_output: If True, captures and logs stdout and stderr of the task .sh script or .py code.
  • python_venv_name: If set, uses the python virtual environment located under ~/.venv/
  • max_files_to_upload: Maximum number of task files to upload to worker from the files_path. Please increase with caution, and prefer to place larger number of files to: 1) on my/shared folders or 2) on git and pull after worker is started.
Returns

Tuple of (Worker that was created to run the task, task success boolean)
